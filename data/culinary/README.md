# Culinary carousel images

The top carousel (under the site name and “Scroll”) uses **kitchen utensils and tools**. By default it loads images from Unsplash. You can replace them with your own by adding files here and updating `index.html` to use local paths.

Suggested filenames (one image per slide):

| Filename       | Subject              |
|----------------|----------------------|
| whisk.jpg      | Whisk                |
| mixer.jpg      | Stand mixer / KitchenAid |
| blender.jpg    | Blender              |
| tongs.jpg      | Tongs                |
| spatula.jpg    | Spatula              |
| scissors.jpg   | Kitchen scissors     |
| knife.jpg      | Chef knife           |
| ladle.jpg      | Ladle or wooden spoon |

Use high-quality, well-lit photos (e.g. 1200×514 or similar 21:9 aspect) for best results. Keep filenames lowercase; update the `src` in each `.carousel-slide` in `index.html` to `data/culinary/whisk.jpg` etc. if you switch to local images.
