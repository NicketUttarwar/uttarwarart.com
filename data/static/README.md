# Static assets

Images used in the about section (culture blocks). All are stored locally in `data/static/`:

- **vada-pav.jpg** — Mumbai (vada pav street food)
- **beef-wellington.jpg** — London (beef Wellington)
- **sourdough.jpg** — San Francisco (sourdough bread)

To replace any image, overwrite the file or add a new one and update the corresponding `src` and `--bg-image` in `index.html`.
